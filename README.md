# CatClaw Desktop

A desktop companion for Claude Code & Codex.

## Downloads

Go to [Releases](https://github.com/luzhipeng728/CatClaw-releases/releases) to download the latest version.

| Platform | Architecture | File |
|----------|-------------|------|
| macOS (Apple Silicon) | M1/M2/M3/M4 | `CatClaw-*-arm64.dmg` |
| macOS (Intel) | x86_64 | `CatClaw-*-x64.dmg` |
| Windows | x64 | `CatClaw-*-Setup.exe` |

## What's included

- **Bun runtime** — bundled, no install needed
- **Claude Code CLI** — bundled
- **Git** — bundled on Windows (macOS uses system git)

## Requirements

- macOS 12+ or Windows 10+
- Claude Code API key (configure on first launch)
